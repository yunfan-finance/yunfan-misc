# Yunfan's Finance & Macro Notes

这是一个记录我对金融市场、个股分析以及宏观世界演进看法的个人知识库。

## 目录结构说明

- **`stocks/`** (个股分析)
  - `A-share/`: A股市场个股分析
  - `HK-share/`: 港股市场个股分析
  - `US-share/`: 美股市场个股分析
- **`industries/`** (行业研究)
  - 记录对各行业的深度分析与跟踪，如AI、新能源、消费等。
- **`macro/`** (宏观与世界演进)
  - `world-evolution/`: 记录我对未来世界格局、技术演进、社会变化的长线思考。
- **`templates/`** (模板)
  - 存放用于个股分析或行业研究的 Markdown 模板文件，保持记录格式的一致性。

## 写作原则

1. **客观独立**：保持独立思考，不盲从市场情绪。
2. **长线思维**：关注事物的本质和长期演进规律。
3. **定期复盘**：对过往的判断进行回顾和修正。
